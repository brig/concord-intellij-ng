Reports invalid values in Concord YAML files.

Some Concord YAML keys expect values of a specific type or format. Invalid values may cause the flow to fail validation or behave incorrectly at runtime. This inspection highlights values that do not match the expected type or constraints so you can fix them before running a flow. 

retry:
  times: "three"   # expected a number

This inspection is applied only to Concord YAML files.
