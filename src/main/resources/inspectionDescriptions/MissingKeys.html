Reports missing mandatory keys in a Concord YAML file.

Certain top-level keys are required for Concord to interpret a flow definition correctly. If a mandatory key is missing, the flow may fail to load or run as expected. This inspection highlights missing keys and provides quick fixes when possible. 

# Missing required items for loop
flows:
  default:
    - task: http
      loop:
         mode: parallel

This inspection is applied only to Concord YAML files.
