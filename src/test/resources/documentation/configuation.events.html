events

Type: object

Configures process event recording and truncation to prevent large or sensitive values from being stored in events

Keys:

  • inVarsBlacklist (string[]) — list of input variable names to exclude from recording
  • metaBlacklist (string[]) — list of metadata keys to exclude from recording
  • outVarsBlacklist (string[]) — list of output variable names to exclude from recording
  • recordEvents (boolean) — enable or disable recording of process events
  • recordTaskInVars (boolean) — enable or disable recording of input variables in task calls
  • recordTaskMeta (boolean) — record task metadata in events
  • recordTaskOutVars (boolean) — enable or disable recording of output variables in task calls
  • truncateInVars (boolean) — truncate recorded input variables
  • truncateMaxArrayLength (integer) — maximum array length before truncation
  • truncateMaxDepth (integer) — maximum object nesting depth before truncation
  • truncateMaxStringLength (integer) — maximum string length before truncation
  • truncateMeta (boolean) — truncate recorded metadata
  • truncateOutVars (boolean) — truncate recorded output variables
