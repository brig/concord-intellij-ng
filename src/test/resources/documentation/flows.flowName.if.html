if

Type: object

Conditional step. Evaluates an expression and executes 'then' or 'else' branches

Keys:

  • else (object[]|string[]) — steps to execute when condition is false
  • if (expression, required) — concord expression (e.g. ${myVar})
  • meta (object) — step metadata
  • then (object[]|string[], required) — steps to execute when condition is true

Example:

- if: ${condition}
  then:
    - log: "true branch"
  else:
    - log: "false branch"
