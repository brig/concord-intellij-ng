concord

Schema for the concord task - start/fork processes, manage API keys

Input Parameters:

  • action (enum, required) — The action to perform
    • start — Start a new process
    • startExternal — Start a process on an external Concord server
    • fork — Fork the current process
    • kill — Kill a running process
    • createApiKey — Create an API key for a user
    • createOrUpdateApiKey — Create or update an API key for a user

Additional parameters depend on the value of action

Output Parameters:

  • ok (boolean, required) — Whether the operation succeeded
  • id (string) — Instance ID (for start) or API key ID (for createApiKey)
  • ids (string[]) — Instance IDs (for fork with multiple processes)
  • name (string) — API key name
  • key (string) — The actual API key value
  • result (string) — Result status (CREATED or UPDATED for createOrUpdateApiKey)
  • expiredAt (string) — API key expiration timestamp
