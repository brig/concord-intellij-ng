multiKeyTask

a task with multiple discriminator keys

Input Parameters:

  • action (enum, required) — action to perform
    • process
    • upload
  • mode (enum) — execution mode
    • sync
    • async

When action = process, mode = sync:

  • timeout (integer) — timeout in seconds

When action = upload:

  • destination (string) — upload destination path

Output Parameters:

  • ok (boolean, required) — whether the operation succeeded
