cron

Type: object

Cron trigger definition. Starts a process on a schedule

Keys:

  • activeProfiles (string[]) — list of profiles that Concord applies for the process
  • arguments (object) — additional parameters that are passed to the flow
  • entryPoint (string, required) — name of the flow that Concord starts when the trigger fires
  • exclusive (object) — exclusive execution configuration for the process. Available only for processes running in projects
  • runAs (object) — username to run the process as when the cron trigger fires
  • spec (cron, required) — cron expression that defines when the trigger fires
  • timezone (string)
