exclusive

Type: object

Exclusive execution configuration for the process. Available only for processes running in projects

Keys:

  • group (string) — exclusive group name used to scope mutual exclusion between processes
  • groupBy (string) — grouping key for exclusive execution. Allowed values: branch name
  • mode (string) — conflict resolution mode (default: cancel)
    • cancel — Cancel all new processes if there is already a running process in the same group
    • cancelOld — Cancel older running processes in the same group when a new process starts
    • wait — Allow only one process in the same group to run New processes wait until the current one finishes
