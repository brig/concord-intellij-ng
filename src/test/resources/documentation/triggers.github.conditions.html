conditions

Type: object

Conditions for GitHub event matching

Keys:

  • branch (regexp) — branch name (default: current repository branch)
  • files (object) — list of file path patterns to match changed files in the event
  • githubHost (regexp) — gitHub host name
  • githubOrg (regexp) — gitHub organization name (default: current repository organization)
  • githubRepo (regexp) — gitHub repository name (default: current repository name)
  • payload (object) — key value conditions that must match fields in the GitHub event payload
  • repositoryInfo (object[]) — list of repositoryInfo entries for matching events that have registered Concord repositories
  • sender (regexp) — event sender name
  • status (regexp) — pull request status name. For pull_request events typical values are opened or closed
  • type (string, required) — gitHub event name
