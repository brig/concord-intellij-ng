conditions

Type: object

Conditions for GitHub event matching

Keys:

  • branch (string) — branch name or regex. Default: is the current repository branch
  • files (object) — list of file path patterns to match changed files in the event
  • githubHost (string) — gitHub host name or regex
  • githubOrg (string) — gitHub organization name or regex. Default: is the current repository organization
  • githubRepo (string) — gitHub repository name or regex. Default: is the current repository name
  • payload (object) — key value conditions that must match fields in the GitHub event payload
  • repositoryInfo (object[]) — list of repositoryInfo entries for matching events that have registered Concord repositories
  • sender (string) — event sender name or regex
  • status (string) — pull request status name or regex. For pull_request events typical values are opened or closed
  • type (string, required) — gitHub event name
