files

Type: object

List of file path patterns to match changed files in the event

Keys:

  • added (regexp|regexp[]) — match events that add files with paths matching the specified patterns
  • any (regexp|regexp[]) — match events that change any files with paths matching the specified patterns
  • modified (regexp|regexp[]) — match events that modify files with paths matching the specified patterns
  • removed (regexp|regexp[]) — match events that remove files with paths matching the specified patterns
