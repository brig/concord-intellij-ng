call

Type: object

Call step. Invokes another flow by name, optionally passing input/output parameters.

Keys:

  • call (string, required) — flow name to call
  • error (object[]|string[]) — error handling steps
  • in (expression|object) — input parameters
  • loop (object) — loop configuration
  • meta (object) — step metadata
  • name (string) — step display name
  • out (string|object|string[]) — output variable mappings
  • retry (object) — retry configuration

Example:

- call: "myFlow"
  in:
    name: "value"
