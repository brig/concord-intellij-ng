github

Type: object

GitHub trigger definition. Starts a process when a matching GitHub event is received

Keys:

  • activeProfiles (string[]) — list of profiles that Concord applies for the process
  • arguments (object) — additional parameters that are passed to the flow
  • conditions (object, required) — conditions for GitHub event matching
  • entryPoint (string, required) — name of the flow that Concord starts when GitHub event matches conditions
  • exclusive (object) — exclusive execution configuration for the process. Available only for processes running in projects
  • ignoreEmptyPush (boolean) — if true, skip empty push notifications where after and before commit IDs are the same (default: true)
  • useEventCommitId (boolean) — if true, use the event commit ID to start the process
  • useInitiator (boolean) — if true, set the process initiator to the event sender
  • version (integer, required) — gitHub trigger version
