form

Type: object

Form step. Displays a form to collect user input and suspends the process

Keys:

  • fields (expression|object[]) — form field definitions
  • form (string, required) — form name
  • runAs (object) — user to run the form as
  • saveSubmittedBy (boolean) — save the user who submitted the form
  • values (object) — preset form field values
  • yield (boolean) — whether to yield the form

Example:

- form: "myForm"
  yield: true
