git

Type: object

Git import definition. Imports a remote Git repository into the process working directory

Keys:

  • dest (string) — destination directory relative to the process working directory (default: ./concord/)
  • exclude (regexp[]) — list of regular expression patterns to exclude files during import
  • name (string) — organization and repository name, e.g. walmartlabs/concord. Automatically expanded to a full URL based on server configuration. Mutually exclusive with url
  • path (string) — path in the repository to copy from
  • secret (object) — reference to a KEY_PAIR or USERNAME_PASSWORD secret. Must be a non-password protected secret
  • url (string) — repository URL in http(s) or git@ format. Mutually exclusive with name
  • version (string) — branch, tag, or commit ID to use (default: master)
