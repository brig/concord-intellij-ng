action

Type: string|expression

The action to perform

Values:

  • start (string) — start a new process
  • startExternal (string) — start a process on an external Concord server
  • fork (string) — fork the current process
  • kill (string) — kill a running process
  • createApiKey (string) — create an API key for a user
  • createOrUpdateApiKey (string) — create or update an API key for a user
