staging

Type: object

Profile entry. Overrides or extends the root configuration, flows, and forms for this profile

Keys:

  • configuration (object) — profile-specific configuration. Overrides root configuration values when this profile is active
  • flows (object) — flow definitions. Each key is a flow name mapping to a list of steps
  • forms (object) — declares web-based forms used by flows to collect user input. Each form is defined under a name; the name is used to reference the form from flows and as the object that stores submitted field values.

    Form definitions can be reused within the same process, and fields can also be defined dynamically at runtime.

    Note: form names may contain only alphanumerics, whitespaces, underscores (_) signs
