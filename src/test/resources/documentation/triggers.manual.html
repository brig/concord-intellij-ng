manual

Type: object

Manual trigger definition. Starts a process only when explicitly requested via UI or API

Keys:

  • activeProfiles (string[]) — list of profiles that Concord applies for the process
  • arguments (object) — additional parameters that are passed to the flow
  • entryPoint (string, required) — name of the flow that Concord starts when the trigger is executed
  • exclusive (object) — exclusive execution configuration for the process. Available only for processes running in projects
  • name (string) — trigger name used for display and identification
