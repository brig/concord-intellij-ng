fields

Type: expression|object[]

Form field definitions

Keys:

  • allow (any) — allowed values (literal/array/object or expression)
  • inputType (string) — input type (for example: text, password, email)
  • label (string) — human-readable label
  • max (integer) — maximum value
  • min (integer) — minimum value
  • pattern (regexp) — regular expression to validate the value
  • placeholder (string) — short hint shown in the input
  • popupPosition (string) — popup position
  • readOnly (boolean) — make the field read-only
  • search (boolean) — allow searching in dropdown input
  • type (string, required) — field type: string, int, decimal, boolean, file (cardinality: ?, +, *)
  • value (any) — default value expression, evaluated when the form is called
