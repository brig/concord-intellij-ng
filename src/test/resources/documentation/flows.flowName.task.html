task

Type: object

Task step. Executes a registered task plugin (e.g. http, smtp, docker)

Keys:

  • error (object[]|string[]) — error handling steps
  • ignoreErrors (boolean) — continue execution on error
  • in (expression|object) — input parameters
  • loop (object) — loop configuration
  • meta (object) — step metadata
  • name (string) — step display name
  • out (string|object) — output variable mappings
  • retry (object) — retry configuration
  • task (string, required) — task plugin name

Example:

- name: "Download artifact"
  task: "http"
  in:
    url: "https://example.com"
  out: response
