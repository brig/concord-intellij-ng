expr

Type: object

Expression step. Evaluates a single expression

Keys:

  • error (object[]|string[]) — error handling steps
  • expr (expression, required) — concord expression (e.g. ${myVar})
  • meta (object) — step metadata
  • name (string) — step display name
  • out (string|object) — output variable mappings

Example:

- expr: ${myVar.method()}
  out:
    result: ${result}
