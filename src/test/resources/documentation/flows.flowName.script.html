script

Type: object

Script step. Executes an inline or external script (e.g. Groovy, JavaScript).

Keys:

  • body (string) — inline script body
  • error (object[]|string[]) — error handling steps
  • in (expression|object) — input parameters
  • loop (object) — loop configuration
  • meta (object) — step metadata
  • name (string) — step display name
  • out (string|object) — output variable mappings
  • retry (object) — retry configuration
  • script (string, required) — script engine name

Example:

- script: groovy
  body: |
    def x = execution.getVariable("myVar")
