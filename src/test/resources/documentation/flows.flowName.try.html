try

Type: object

Try step. Executes steps with error handling support.

Keys:

  • error (object[]|string[]) — error handling steps
  • loop (object) — loop configuration
  • meta (object) — step metadata
  • name (string) — step display name
  • out (string|string[]) — output variable mappings
  • try (object[]|string[], required) — steps to execute with error handling

Example:

- try:
    - log: "Risky operation"
  error:
    - log: "Error occurred"
