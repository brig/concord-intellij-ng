exclusive

Type: object

Ensures only one process runs per project + group. Conflict behavior is controlled by mode

Keys:

  • group (string, required) — name of the exclusivity group
  • mode (string) — conflict resolution mode (default: cancel)
    • cancel — New submission is immediately cancelled if another run in the same project+group is running
    • cancelOld — Running process in the same project+group is cancelled in favor of the new submission
    • wait — Enqueue: only one run per group; others wait

Example:

configuration:
  exclusive:
    group: "myGroup"
    mode: "cancel"
