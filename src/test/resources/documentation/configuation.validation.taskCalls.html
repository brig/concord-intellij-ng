taskCalls

Type: object

Validation settings for task call input and output parameters

Keys:

  • in (string) — validation mode for task input parameters
    • disabled — no validation
    • warn — log warnings but continue
    • fail — fail on validation errors
  • out (string) — validation mode for task output parameters
    • disabled — no validation
    • warn — log warnings but continue
    • fail — fail on validation errors
