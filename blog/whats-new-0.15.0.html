What’s New in Concord IntelliJ 0.15.0

January 14, 2026 · One min read
Brig
Concord IntelliJ Plugin Maintainer

The 0.15.0 release introduces semantic syntax highlighting for Concord YAML files. This update transforms the way you read and write Concord flows by making the structure and logic of your automation code visually distinct.

Semantic Syntax Highlighting​

IDE Compatibility​

This release also updates the plugin to support IntelliJ IDEA 2025.3.1. We recommend updating your IDE to ensure the best performance and compatibility with the new highlighting features.
